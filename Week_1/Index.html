A list of all Html Tags and their Functions

  • <!DOCTYPE >: Defines the document type and version of HTML being used.
  • <html >, </html > : The root element that wraps the entire HTML document.
  • <head>,< /head >: Contains meta-information about the document, such as title, character encoding, and linked stylesheets.
  • <title>, </title >: Sets the title of the web page, displayed in the browser's title bar or tab.
  • <meta>, </meta >: Provides metadata about the document, such as character encoding or author information.
  • <link>,< /link >: Links external resources like stylesheets or icon files to the HTML document.
  • <style >, </style >: Embeds CSS (Cascading Style Sheets) within the HTML document.
  • <script>, </script >: Embeds JavaScript code or links to external JavaScript files.
  • <body>,< /body >: Contains the main content of the web page, including text, images, and other elements.
  • <header>, </header >: Represents introductory content or a container for navigational elements like logos and menus.
  • <nav> </nav>: Defines a section of navigation links.
  • <main> </main >: Contains the main content of the document.
  • <section>, </section >: Represents a thematic grouping of content within the document.
  • <article>, < /article>: Defines a self-contained piece of content, such as a news article or blog post.
  • <aside>,</aside>: Represents content that is tangentially related to the content around it.
  • <footer> </footer>: Contains footer information, often including copyright or contact information.
  • <h1>, h2, h3, h4, h5, h6 >: Define headings of different levels, with < h1 > being the highest level and < h6 > the lowest.
  • <p>< /p>:Represents a paragraph of text.
  • <a>< /a>: Creates hyperlinks to other web pages or resources.
  • <img>: Embeds images on the web page.
  • <ul>: Defines an unordered list.
  • <ol>: Defines an ordered (numbered) list.
  • <li>: Represents a list item within <ul> or < ol >.
  • <div>: A generic container for grouping and styling elements.
  • <span>: A generic inline container for styling or scripting.
  • <table>: Defines a table.
  • <tr>: Represents a table row.
  • <th>: Defines a table header cell.
  • <td>: Represents a table data cell.
  • <form>: Creates a form for user input.
  • <input>: Defines an input field within a form.
  • <textarea>: Defines a multiline text input field.
  • <utton>: Creates a clickable button within a form.
  • <label>: Represents a label for an input field.
  • <select>: Creates a dropdown list within a form.
  • <option>: Defines an option within a
  • <select> element.
  • <iframe>: Embeds an inline frame for displaying external content.
  • <audio>: Embeds audio content on the web page.
  • <video>: Embeds video content on the web page.
  • <canvas>: Provides a drawing surface for dynamic graphics and animations.
